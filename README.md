# student-grade-prediction-ml
A machine learning project that predicts student marks based on grades using regression techniques, with performance evaluation and visualization.
# 🎓 Student Grade Prediction using Machine Learning

This project demonstrates how to predict student marks based on their grades using Machine Learning regression techniques.

---

## 🚀 Features
- Data preprocessing
- Model training using regression
- Performance evaluation:
  - MAE (Mean Absolute Error)
  - MSE (Mean Squared Error)
  - RMSE (Root Mean Squared Error)
  - R² Score
- Visualization using matplotlib

---

## 🧠 Technologies Used
- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
